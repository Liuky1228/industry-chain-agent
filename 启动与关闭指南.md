## 产业链分析系统 — 启动与关闭指南

本指南面向零基础用户，教你如何在自己的电脑上启动和关闭「产业链智能分析系统」的前端和后端服务。

---

### 一、基本概念

这个项目由两部分组成：

**后端**（Backend）：负责数据处理、AI 分析、生成报告等核心逻辑。运行在 **8004** 端口，使用 Python + FastAPI 框架。

**前端**（Frontend）：负责网页界面展示，让你能在浏览器中操作。运行在 **3004** 端口，使用 React + Vite 框架。

两者必须**同时运行**，系统才能正常使用。

### 三、启动服务

#### 方式一：使用 start.bat（推荐）

项目根目录下有一个 `start.bat` 文件，双击即可一键启动前后端。

1. 打开文件资源管理器，进入项目根目录
2. 双击 `start.bat`
3. 会弹出两个命令行窗口（一个后端、一个前端），**不要关闭它们**
4. 等待几秒，看到类似以下信息表示启动成功：
   - 后端窗口显示：`Uvicorn running on http://0.0.0.0:8004`
   - 前端窗口显示：`Local: http://localhost:3004/`
5. 打开浏览器，访问 `http://localhost:3004` 即可使用

#### 方式二：手动分别启动

如果 start.bat 不可用，可以手动启动。需要打开**两个命令行窗口**。

**启动后端（第一个窗口）：**

1. 按 `Win + R`，输入 `cmd`，回车，打开命令提示符
2. 输入以下命令，进入项目目录：
   ```
   cd /d C:\Users\kingdee\.qoderworkcn\workspace\mqok2n7tcxh99ymk\outputs\industry-chain-agent
   ```
3. 输入以下命令启动后端：
   ```
   .\venv\Scripts\python.exe -m uvicorn app.main:app --host 0.0.0.0 --port 8004 --reload --reload-dir app
   ```
4. 看到 `Uvicorn running on http://0.0.0.0:8004` 表示成功
5. **保持此窗口打开，不要关闭**

**启动前端（第二个窗口）：**

1. 再打开一个新的命令提示符窗口
2. 输入以下命令进入前端目录：
   ```
   cd /d C:\Users\kingdee\.qoderworkcn\workspace\mqok2n7tcxh99ymk\outputs\industry-chain-agent\frontend
   ```
3. 输入以下命令启动前端：
   ```
   npm run dev
   ```
4. 看到 `Local: http://localhost:3004/` 表示成功
5. **保持此窗口打开，不要关闭**

### 五、关闭服务

#### 方式一：直接关闭命令行窗口

最简单的方法：直接把启动后端和前端的两个命令行窗口关掉即可。

#### 方式二：使用快捷键（推荐）

在每个命令行窗口中按 `Ctrl + C`，服务会优雅停止。

#### 方式三：端口被占用时强制关闭

如果重启时提示「端口已被占用」，说明上次的进程没有完全退出。按以下步骤操作：

1. 打开命令提示符（`Win + R` → 输入 `cmd` → 回车）
2. 查找占用 8004 端口的进程：
   ```
   netstat -ano | findstr :8004
   ```
3. 记下最后一列的数字（PID），例如 `15444`
4. 强制结束该进程：
   ```
   taskkill /PID 15444 /F
   ```
5. 如果仍有子进程占用，查找子进程：
   ```
   wmic process where "ParentProcessId=15444" get ProcessId
   ```
6. 对每个子进程 PID 也执行 `taskkill /PID xxx /F`
7. 前端 3004 端口同理，把上面的 `8004` 换成 `3004` 即可

### 六、常见问题

**Q: 启动后端时报错 "Address already in use"（端口被占用）**
A: 参考上面「方式三」强制关闭占用端口的进程，然后重新启动。

**Q: 启动前端时报错 "npm 不是内部或外部命令"**
A: 需要先安装 Node.js。去 https://nodejs.org 下载 LTS 版本安装，安装后重新打开命令提示符。

**Q: 浏览器打开 localhost:3004 显示空白或报错**
A: 先确认后端是否已启动（浏览器访问 `http://localhost:8004/docs`，如果能看到 API 文档页面说明后端正常）。如果后端没启动，先启动后端。

**Q: 分析到一半想停止怎么办？**
A: 在任务进行中的页面上点击红色的「终止任务」按钮即可。已下载的研报数据会保留。

**Q: 关闭电脑后需要重新启动吗？**
A: 是的。每次开机后都需要重新启动前后端服务才能使用系统。
